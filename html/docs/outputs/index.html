These processors can only be used in the Output section.

Xml

Output in XML format.

Example:

output:
  - xml
    destination:
      - www.foo.com
      - www.bar.com

Email

Output in email format.

Example:

output:
  - email
    format: json
    destination:
      - foo@foo.com
      - bar@foor.com

Html

Output in HTML format.

Example:

output:
  - html
    destination:
      - www.foo.com
      - www.bar.com

Image

Output in an image format. The data fed into the output can be a URL (must start with http) or an input filename.

Example:

output:
  - image
    destination:
      - www.foo.com
      - www.bar.com

Json

Output in JSON format.

Example:

output:
  - json
    destination:
      - www.foo.com
      - www.bar.com

Plain

Output in plain text.

Example:

output:
  - plain
    destination:
      - www.foo.com
      - www.bar.com

Text

Output in text format.

Example:

output:
  - text
    destination:
      - www.foo.com
      - www.bar.com
